interface Cached<T>

Created by Brian Parma on 2/10/16.

This is basically just the lazy delegate copy and pasted, then modified so that it can be reset by setting it equal to null

Represents a value with lazy initialization.

To create an instance use the cached function.

Properties

value

 abstract var value: T?

Gets the lazily initialized value of the current Cached instance. Once the value was initialized it must not change during the rest of lifetime of this Cached instance.
