ProximitySensor

interface ProximitySensor

ProximitySensor is implemented as a boolean-sensor and works based on the lifecycle of the binded activity. It will be automatically unbinded on the activity destroy.

It returns just two values "NEAR" or "FAR". Threshold is done on the LUX value i.e. the LUX value of the light sensor is compared with a threshold. A LUX-value more than the threshold means the proximity sensor returns "FAR". Anything less than the threshold value and the sensor returns "NEAR".
